#  Golf Website UI Project

This is a frontend project where I built an interactive golf-themed website to practice modern UI design and animations.

The main goal of this project was to improve my skills in JavaScript animations and create a smooth user experience using GSAP.


##  What I built

* A landing page with a full-screen video background
* Custom cursor that follows the mouse and changes on hover
* Smooth scroll-based animations using GSAP
* Interactive cards with hover effects
* Continuous scrolling text section
* Clean and modern layout design


##  Tech Used

* HTML
* CSS
* JavaScript
* GSAP (for animations)


##  What I learned

* How to use GSAP and ScrollTrigger for animations
* Handling mouse events and DOM manipulation
* Creating interactive UI elements
* Improving website design and layout


##  Limitations

* Not fully responsive yet (works best on desktop)
* No backend functionality


##  How to run

Just download the files and open `index.html` in your browser.
